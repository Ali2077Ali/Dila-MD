    <h4 class="title-animation">How to create DILAMD</h4>
    <p><strong>1. You must have these things before deploying:</strong></p>
    <ul>
        <li>_GitHub Account._</li>
        <li>_MongoDB URL (sign up and get MongoDB URL)._</li>
        <li>_Host._</li>
    </ul>
    <p><strong>2. Deploy steps:</strong></p>
    <ul>
        <li>_Fork the DILA MD repository._</li>
        <li>_Link with your WhatsApp using Scan QR code or Pair code._</li>
        <li>_Open `config.js` on your forked repository. Add `SESSION_ID` and `MONGODB_URL`_</li>
        <li>_Deploy using your host._</li>
    </ul>
